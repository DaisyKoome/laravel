Crucial laravel commands

To create a new laravel project
laravel new firstproject
composer create-project --prefer-dist laravel/laravel firstproject

To start the development server on the browser (display on localhost)
php artisan serve

To stop serving (stop running the development server on localhost)
Press CTRL + Z

To run the development server on a different port than the default 8000
cd into the specific laravel project folder
php artisan serve --port=8081
